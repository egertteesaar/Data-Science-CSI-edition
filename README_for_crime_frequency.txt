To run crime_frequency_visualization.ipynb:

1) Install and run Jupyter Notebook
2) Open "crime_frequency_visualization.ipynb"
3) Make sure you have OpenCV python interface installed ("pip install opencv-python")
4) Make sure you have pandas, numpy and matplotlib available in running environment
5) Run the code in notebook

The code in the file will:

1) Color the municipalities on the Estonian map to visualize the property crime rates in them.
2) Show a histogram about the distribution of crime rates.
3) List the top 5 places with the highest and lowest crime rates.
4) Show a scatter plot of population density against crime rates and their correlation cofficient.
