Software, that is required for successfully running Visualising Crime on Estonia's Map.ipynb.

Have Jupyter Notebooks installed with Anaconda3 and python3.7 (Anaconda version 3-5.3.1-Linux-x86_64)
Additionally:

conda install -c anaconda pillow
conda install -c anaconda basemap
conda install -c anaconda basemap-data-hires
conda install -c ioos geopandas

pip install --user shapelib

sudo apt install libsqlite3-dev #for shape files
